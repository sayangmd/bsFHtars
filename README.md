# 项目简介

## 核心功能
- 用户模块：
  - 注册：账号密码，头像以及性别等，基本信息填写。
  - 登录：账号密码登录，登录异常时触发安全提醒，需验证短信确认身份。
  - 个人信息管理：编辑个人资料，管理收货地址，修改密码、更换手机号。
  - 消息通知：系统自动推送各类消息至公告，可查看详情。
- 订单模块：
  - 交易订单生成：用户发起资源置换或购买请求后，系统自动生成订单，包含订单编号、资源详情及支付。
  - 订单状态跟踪：订单支付与否，订单历史查询，订单增删查看功能。
- 收藏模块：
  - 收藏：用户可对感兴趣的资源点击收藏，批量取消收藏。
  - 资源：资源的增删改查操作，评论及查看评论。
- 资源模块：
  - 发布新资源：用户可以交易自己的资源。
- 管理员模块：
  - 用户管理：管理员查看用户列表，包含基本信息、注册时间。
  - 资源审核：审核用户发布的闲置资源，包括图片真实性、描述合规性、类别准确性。
  - 资源的增删改查：管理员发布推荐资源，编辑已发布资源。
  - 推荐：基于用户浏览历史、收藏记录智能推荐相似资源。
  - 系统公告发布：管理员发布平台公告，公告置顶于首页通知栏，支持富文本编辑。

## 适用场景
- 校园内学生之间短期闲置资源的置换和购买。
- 促进校园资源共享，减少浪费。

## 技术特点
- 采用Spring Boot框架，易于开发和部署。
- 具备良好的扩展性，支持后续功能模块的添加。
- 使用数据库文件进行数据存储，保证数据安全。
- 支持消息通知和订单状态跟踪，提高用户使用体验。
